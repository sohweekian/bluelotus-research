## Core Mathematical Framework

### Bayesian Thesis Updating (NITE-PEI Engine)

$$P_{posterior} = \frac{P_{prior} \times LR_{adj}}{P_{prior} \times LR_{adj} + (1 - P_{prior})}$$

$$LR_{adj} = 1 + (LR_{raw} - 1) \times (1 - noise\_discount)$$

### Kill Condition Risk Index (CKRI)

$$CKRI = \frac{\sum w_i \cdot P_{kill,i} + \lambda \cdot n_{corr}}{\sum w_i}$$

| Zone | CKRI Range | Action |
|------|-----------|--------|
| CLEAR | 0.00 – 0.20 | Normal operations |
| WATCH | 0.20 – 0.40 | Heightened monitoring |
| ELEVATED | 0.40 – 0.60 | Review position sizing |
| HIGH | 0.60 – 0.80 | De-risk equities via equity gate |
| CRITICAL | 0.80 – 1.00 | Full review, manual CIO decision |

### Kelly-NITE Position Sizing

$$f^* = \frac{b \cdot p - q}{b} \times f_{kelly} \times coherence$$

$$coherence = 1 - \left(H_{norm} \times 0.5 + D_{branch} \times 0.5\right)$$

where $H_{norm}$ is Shannon entropy normalized and $D_{branch}$ is PEI scenario branch dispersion.

### 8-Lens Quantitative Score

$$S_{stock} = \sum_{i=1}^{8} w_i \cdot L_i \quad \text{where} \quad \sum w_i = 1, \quad L_i \in [0, 5]$$
